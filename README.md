# MolecularCagesGeneration

This repository hosts the MolecularCagesGeneration project. Its main focus is the analysis and development of tools around molecular cage construction.

## Repository Layout
- **InterconnectionTreesEnumeration/** – contains the `Gen_ICT` algorithm and utilities used to analyze its behavior and performance.

Each folder includes its own README with more detailed, folder-specific guidance.

## Next Steps
This codebase is in preparation. As the project matures, additional documentation, examples, and polishing passes will be added here to support that release.
